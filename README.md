# Retail Sales Excel Dashboard

## Project Overview
Built an interactive retail sales dashboard using Microsoft Excel.

## Tools Used
- Excel
- Pivot Tables
- Pivot Charts
- Slicers

## KPIs
- Total Sales
- Total Orders
- Total Quantity Sold
- Average Sales
- Average Discount

## Dashboard Features
- Sales Trend Analysis
- Sales by Region
- Sales by Category
- Sales by Channel
- Customer Segmentation

## Key Insights 
- Central region generated the highest sales.
- Online channel contributed significantly to revenue.
- Sports category was among the top-performing categories.
